如果你已經有想要呈現的色彩風格的色碼，也可以使用官方的vscode-theme-generator

使用與安裝很簡單，只需先下載官方的原始碼：

git clone https://github.com/Tyriar/vscode-theme-generator-quick-start
cd vscode-theme-generator-quick-start
npm install
